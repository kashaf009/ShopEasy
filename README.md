# 🛒 E-Commerce Website (Electronics)

A modern and responsive **eCommerce web application** built using **React.js** and **Tailwind CSS**, designed for showcasing electronic products like **laptops, mobiles, and tablets**.

## 📌 Features

- 📱 Browse electronic products (Laptops, Mobiles, Tablets)
- 🔍 Search and filter products
- 🛒 Add to cart functionality
- ⚡ Fast and responsive UI
- 🎨 Clean and modern design with Tailwind CSS
- 🔄 Component-based architecture using React

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **State Management:** Context API / Redux
